### Tooltip Module

#### Usage
This is data driven based on ```data-tooltip``` and it's options. The FP tooltip module has the following options:
````
var _defaults = {
    event: 'hover',
    direction: 'top',
    xOffset: '20px',
    yOffset: '20px',
    url: '',
    selector: 'tooltip-content'
};
````
